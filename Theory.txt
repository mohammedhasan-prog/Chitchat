Added the Dashboard UI based on the provided dashboard HTML design. Included total message sent, active chat counts, and an Avatar upload feature that pushes to the /upload/?is_profile_pic=true endpoint. Fixed a bug where uploading a profile picture in the dashboard did not immediately reflect the image change in the rest of the application. This was resolved by passing an onUpdateUser={setUser} prop to the Dashboard component and triggering it after the frontend received the newly generated picture url from the backend.
